📋 Overview
JUICIFY is a modern e-commerce platform for premium cold-pressed juices. Built with React and Tailwind CSS, it offers a seamless shopping experience with user authentication, cart management, and secure checkout.

Live Demo: https://pennyjuicemodifiedusjuicifyreacttai.vercel.app/

✨ Key Features
🛍️ Shopping
9+ organic juice flavors with detailed product pages

Filter by category (Citrus, Tropical, Berry, etc.)

Sort by price or name

3x3 responsive product grid

👤 User Features
Signup/Login with localStorage persistence

Shopping cart with quantity controls

Persistent cart across sessions

Protected checkout (login required)

Email confirmations via Web3Forms

🎨 Design
Fully responsive (mobile, tablet, desktop)

Clean orange-branded UI

Mobile hamburger menu with sidebar

Loading states & toast notifications

🛠️ Tech Stack
Frontend	Styling	Build Tools	Data

React Router	React Icons	ESLint	Web3Forms
Context API	Custom CSS	PostCSS	-

🚀 Getting Started
bash
# Clone & install
git clone https://github.com/joheaven7/penny-juice-react-tailwind.git
cd penny-juice-react-tailwind
npm install

# Run locally
npm run dev

# Build for production
npm run build
